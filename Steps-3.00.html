Microsoft does not have a cloud specifically titles Power BI Cloud, instead Microsoft has a separate cloud Service called SQL Azure that can be purchased independently or used in conjunction with Power BI.  There are many advantages and perks when using a cloud service.

            Migrate massive databases and scale up to 100 TB

  • You can get rapid, dynamic storage with SQL Database Hyperscale, the highly performant database scaling technology. Hyperscale allows SQL Database to adapt to your workloads and optimize resources on demand. Microsoft can provide one or more additional compute nodes that can serve your read-only workload so businesses can use them as a hot standby in se of failover.
  • You can now perform operations in constant time, regardless of the size of the data operation.
  • It is now possible to scale compute and storage resources rapidly and independently, without sacrificing performance.

Optimize performance with built-in intelligence

Improve memory usage, throughput, latency, and query performance.

  • Intelligent performance tuning uses built-in intelligence with machine learning and adaptive technologies to continuously optimize database performance in real time. You can now automatically receive tuning recommendations, insights, and performance improvements.
  • Intelligent query processing also improves workload performance with minimal implementation effort. Some features include adaptive query processing, which adapts optimization strategies to application workload runtime conditions, and approximate query processing, designed to aggregate large data sets when responsiveness is critical.

       Reduce the burden of data-tier management and save time and costs by migrating workloads to the cloud.

Use your SQL Server licenses with Software Assurance to pay a reduced rate when migrating to the cloud. Save up to 55 percent with Azure Hybrid Benefit, and up to 80 percent with reserved capacity.
